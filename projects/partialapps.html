Partial Applications

Projects > Programming > Partial Applications

Project | Source

Tools & skills: Clojure, ClojureScript, SASS, Compojure, MongoDB, Memcached, Heroku, Node

Partial Applications is a set of questions, ideas, and aphorisms to inspire creative solutions to difficult programming problems. It’s inspired by Peter Schmidt and Brian Eno’s Oblique Strategies, one of my favorite creative tools. In addition to the web app, it includes a tiny JSON API and a command-line interface.

The site is written in Clojure and ClojureScript. It’s built on top of some wonderful libraries, including Compojure, Hiccup, Monger, Metis, and JayQ. It’s hosted on Heroku and uses a MongoDB backend.
